## 运行命令

```bash
# 开发运行命令
pnpm run dev

# 打包命令
pnpm run build

# 打包后运行命令
pnpm run start

# 基于 pm2 的运行命令
pnpm run pm2

# 单元测试
pnpm run jest

# lint 测试
pnpm run lint

# 代码格式化
pnpm run prettier

# 一键上传代码
pnpm run cz
```
